# Contributing

We are open to, and grateful for, any contributions made by the community. By contributing to stickers-formatter, you agree to abide by the [code of conduct](https://github.com/GlobalTechInfo/mediaforge/blob/main/CODE_OF_CONDUCT.md).


## Commit Messages

Please follow [conventional commits](https://www.conventionalcommits.org/en/v1.0.0/)

## Testing

Please update the tests to reflect your code changes. Pull requests will not be accepted if they are failing on GitHub actions.

## Documentation

Please update the [documentation](https://github.com/GlobalTechInfo/mediaforge#readme) accordingly so that there are no discrepancies between the App and the documentation.
